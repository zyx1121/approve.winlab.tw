# Approve

Sign-off and approval workflow for NYCU WinLab — request, review, and approve with email notifications.

## Tech Stack

- **Framework**: Next.js (App Router)
- **Backend**: Supabase
- **Styling**: Tailwind CSS + shadcn/ui
- **Email**: React Email
- **Package Manager**: Bun

## Getting Started

```bash
bun install
bun dev
```
